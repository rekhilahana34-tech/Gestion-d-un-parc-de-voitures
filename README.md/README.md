Atelier 3 – Gestion d’un parc de voitures



Ce projet a été réalisé dans le cadre de l’atelier 3 de Programmation Orientée Objet (POO). 

L’objectif est de créer une application en Python permettant de gérer un parc de voitures.



Le programme permet de modéliser des voitures à l’aide de classes et d’appliquer les concepts 

de la programmation orientée objet comme les attributs, les méthodes et les objets.

Nom: Hana rekhila 

Cours : Programmation 2
